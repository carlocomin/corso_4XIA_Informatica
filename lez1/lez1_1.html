Lezione 1 — Matrici a due dimensioni in C/C++ (ripasso strutturato)

Obiettivi della lezione

  • Richiamare definizioni e terminologia: righe, colonne, indicizzazione a[r][c], casi quadrati/rettangolari.
  • Comprendere il modello di memoria row-major e le conseguenze su performance e pattern di attraversamento.
  • Ripassare dichiarazione, inizializzazione, I/O, attraversamento per riga/colonna.
  • Passare matrici a funzione in C e C++ in modo corretto.
  • Introdurre e confrontare le rappresentazioni dinamiche: C (malloc/free, VLA C99) e C++ (new/delete, std::vector, flatten 1D, std::vector<std::vector<T>>).
  • Allenarsi su problemi classici (temperature, diagonali, moltiplicazione, pattern di riempimento) e challenge Polibio.

  1. L1.1 (questa): introduzione, obiettivi, convenzioni, prerequisiti.

Prerequisiti (auto-check)

Spunta mentalmente. Se manca qualcosa, lo riprendiamo al volo durante la pratica.

  • Array 1D, cicli for annidati, funzioni con parametri per riferimento/valore.
  • I/O di base (C: scanf/printf · C++: cin/cout).
  • Nozioni base di const/constexpr, differenza stack/heap (cenno per le dinamiche).

Convenzioni di corso & doppia colonna C/C++

Useremo esempi in C++ (prevalente) affiancati dalla versione C quando utile. Dove la sintassi diverge sostanzialmente, mostreremo un confronto side-by-side. 

// C++ (enfasi)
constexpr int R=3, C=5;
int a[R][C]{};             // zero-init
for (int r=0; r<R; ++r)
  for (int c=0; c<C; ++c)
    a[r][c] = r*C + c;     // per riga: migliore locality

// C (ripasso)
#define R 3
#define C 5
int a[R][C] = {0};
for (int r=0; r<R; ++r)
  for (int c=0; c<C; ++c)
    a[r][c] = r*C + c;
